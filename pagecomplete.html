page d'annexes libre service

Ce libre service est destiné à la communauté RPG forumactif, simplifié au possible. Chaque onglet s'affiche au clic, plus d'onglets peuvent bien sûr être ajoutés. Le code est entièrement personnalisable et modifiable, et est proposé essentiellement en tant que base. Il est cependant possible de l'utiliser tel quel.

Le code peut être ajouté comme une page dans les modules forumactif, également compatible avec tumblr (et probablement ailleurs). Il peut probablement être adapté en page d'accueil ou post, mais je recommanderais dans ce cas là d'utiliser directement le libre service de Jessica Stamos dont je me suis servie pour les onglets (le lien se trouve dans les crédits).

Les codes et autres informations d'installation se trouvent dans l'onglet informations. L'onglet exemples et balises sert de démo pour les éléments esthétiques du CSS, comme les titres, cet encadré ou du simple texte.
L'onglet crédits n'est pas à négliger non plus, et il vous est bien sûr demandé de partager ces crédits dans le cas où vous vous servez de ce code comme base.
